# Project 1 – Data Science Introduction

## What is Data Science?
Data Science is the field of extracting insights and knowledge from data using scientific methods, algorithms, and systems.

## What I Learned
- Data Science involves collecting, cleaning, analyzing, and visualizing data.
- It combines skills from statistics, computer science, and domain knowledge.
- Python is one of the most used tools for data science.
